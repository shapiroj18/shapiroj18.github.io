Project Ideas and Scratch Work

A list of random project ideas I've had, some of which I have worked on.

Shareable Bookmarks →

Social Media Platorm that lets users share their web bookmarks with one another.

Spotify Music Discovery, but for Real →

Spotify API application that pulls data from music critic's best music archives.

Best Brewery →

Software built to find the best brewery and/or beer within a certain distance.

Architecture Maps →

Software that will map the architecure of a software project.

CI Cron Automation →

Use scheduled CI runs to automate basic scripted jobs.

Ride Sharing →

Find people nearby who want to go to similar locations to share rideshares with.

Social Radio →

Tune into friends livestreaming radio. They can put on playlists through spotify, talk, etc. Shell/Electron/Tauri app would be fun to experiment with.

Screenshot Helper →

An application that allows you to screenshot web pages and automatically generates (and stores?) a qr code or short url so you can easily access again.

Auto-Compounds →

Software that could automatically generates numbers for references and chemical compounds for papers.

Phish Show Colors →

Software that watches videos of Phish playing live and see if the colors of the lights correlate with the rating of the show or jam on phish.net.

Personal Site →

Site that showcases projects ideas I have and portfolio items.

Snood →

Remake snood to learn a frontend language or framework.

Delayed Texting Service →

You know how there's a send later function for emails? How about that for texting.

Object Positionality →

Find a way to accessibly produce little dots that you can stick to your body, an object, etc that will transmit signals of positionality to software that can convert to positions in real time.

Food Trucks Service →

Build an application that effectively tracks locations of food trucks near you.

Compiler Map →

Compile code into one language, then another, then another based on a map to see what turns up.

Shell Messenger →

Send messages to your friends through the shell.

Radio Decoder →

Decode radio signals directly from software.

Open Source Zapier →

It seems like the workflows for Zapier may be doable to recreate and open source (some of it already is).

Chore Chart →

Individualized email reminders about what chores are due in advance.

Jobs Scraper →

It would be nice to have job alerts for specific organizations you are following. Could look generally at something like Linkedin/Indeed, or pull data from specific org websites.

Git Clone Maps →

Software that will map the cloning history (and locations) of a remote repo.
